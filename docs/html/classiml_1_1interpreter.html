Detailed Description

The interpreter class evaluates the input and returns the result in an output file.

Interpreting is done by a straightforward alghorithm, using a stack to store the tag hierarchy, an unordered_map storing the links between tag attribute names and resulting lists and another stack containing the hierarchy of lists of values to be modificated and returned as a result.
