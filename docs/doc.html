1. Overview

1.1 Introduction

iml is a light weight C++ language interpreter library. It consists of a extremely fast parsing/interpreting mechanism. The iml language is inspired by an uni data structutes project IML. It is easy to develop new tags, and its very easy to integrate and use the library.

IML as a language can be used to evaluate mathematical expressions with the use of tags or to structure arrays which can be used with other programs. iml provides a small range of tags to be played with.

If you want to start writing code as quickly as possuble, here are some examples representing some IML syntax and how to use the interpreter: example.
